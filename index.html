Book Finder

Program Description:
This program is a simple book search application that allows users to search for books based on their tags or other properties.
When the user enters a search term and clicks the search button, the program filters the available books and displays the matching books on the page.

To run the program, follow these steps:
Enter a search term in the input field. You can try various search terms based on the book tags or other properties. For example:

fiction
nonfiction
science fiction
memoir
sociology
anthropology
fantasy
classic
adventure
or book title or author name like :
George Orwell
1984
Shelby Van Pelt
The City We Became




The program consists of several JavaScript files:
bookList.js: This file contains an array of book objects, each with properties like title, author, and tags.
helper.js: This file contains helper functions for flattening object values into an array, structuring book objects as HTML elements, and rendering the book elements to the DOM.
script.js: This file contains the main logic of the program, including functions for capturing the search value, filtering books based on the search input, formatting the filtered books as HTML elements, and handling the search button click event.
